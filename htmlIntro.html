HTML(HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript). Hypertext refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. HTML uses Markup to annotate text, images, and other content for display in a Web browser. HTML markup includes special elements such as <p> tag,<img> tag etc.
